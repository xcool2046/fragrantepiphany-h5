# Frontend（Vite + React）

> 详细说明请参考根目录 `docs/dev-guide.md`。

## 快速开始
```bash
npm install
npm run dev -- --host --port 4173   # http://localhost:4173
```

## 构建与预览
```bash
npm run build
npm run preview
```

## 环境变量
- `VITE_API_BASE_URL`：后端 API 根路径（开发默认 `http://localhost:3000`，生产请填真实域名）。

## Lint
```bash
npm run lint
```

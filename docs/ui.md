# UI 与配色建议（基于 PPT 主题）

## 配色方案（暖杏主题，贴近 PPT）
- 背景浅米：#F7F0E5 / #F7F2ED —— 作为大面积背景。
- 卡片浅暖：#F3E6D7 / #FFFFFF —— 内容承载区。
- 强调色（金杏）：#D4A373 —— CTA、价格、选中、高亮。
- 辅色（玫瑰灰）：#C49BA3 —— 标签/次强调。
- 文字：主文本 #2B1F16，次文本 #6B5542；禁用/占位 #94A3B8。
- 阴影：卡片 `0 12px 30px rgba(0,0,0,0.08)`，按钮轻阴影或无阴影。

## 字体与层级
- 英文：Inter / Source Sans /系统无衬线；中文：更纱黑体 / 思源黑体 / 系统无衬线。
- 标题：加粗，行高 1.2–1.3；正文：常规，行高 1.5。
- 按钮：半径 12–16px，投影弱化或取消以保证 H5 性能。

## 组件与布局建议
- 卡片：浅底（#F3E6D7/#FFF）+ 1px 边框（深色低透明）+ 18–22px 圆角；选中用描边/内阴影代替重投影。
- 抽牌交互：半圆轮盘（210–240°），手势放大/滑动/点击；选中放大 + “放入槽位”，飞入槽位 200–250ms；第三张后自动跳转，过渡 0.6–0.8s。
- 转场：轻量波纹/渐变/少量粒子，避免重度粒子和滤镜；优先保持 60fps。
- 表单/问卷：每问四选一，已选高亮且有确认/下一步按钮；底部进度提示。
- 结果页：分区展示 Past/Now/Action/Future + 推荐；推荐可卡片化（标题 + 一行描述）。

## 适配与可用性
- H5 优先：竖屏第一，横屏和桌面做响应式缩放；按钮高度 ≥ 44px，主要操作固定底部栏。
- 颜色对比：CTA 按钮文本对比度 ≥ 4.5；禁用态降低饱和度与透明度。
- 状态反馈：加载/支付中使用轻量骨架或旋转指示；成功/失败提示用 Toast/气泡，避免遮挡主要内容。

## 页面级 UI 规范（按暖杏主题）
- Home（落地）：浅米背景；Hero 实图（暖杏滤镜）；流程卡片 2×2，卡片浅暖底 + 1px 边框；主 CTA 金杏实底，次 CTA 半透明；语言按钮“EN / CN”浅描边；底部 Tab 半透明浅背景。
- Questionnaire：浅米背景，题干深文字、选项卡浅暖底；选中态描边/填充金杏；底部进度和提交按钮（金杏实底）；Tab 栏同 Home。
- Draw（抽牌）：右侧半圆轮盘深底卡背，左右留白；左侧 Past/Now/Future 槽位浅底描边；顶部提示文案；选中放大 + 按钮金杏；跳转前轻量 Loading；Tab 栏浅背景。
- Result：Past 免费区浅底；付费区（Now/Action/Future/推荐）浅底分区卡片；支付状态条/按钮金杏；可有“再次占卜/分享”次要 CTA；Tab 栏。
- Pay：订单摘要卡片浅底，价格金杏高亮；Stripe CTA 金杏实底；回跳提示文案；Tab 栏。
- 共通：圆角卡片 18–22px、按钮 14–18px；阴影轻；图标 FontAwesome；图片为真实可商用素材（暖杏/米色调）。

  - 区分免费（Past 摘要）与付费（完整解读）视图。
  - 支付成功后自动解锁内容。
- **管理后台**：

  - 简洁的表格视图与表单编辑。

# 文字易读性优化方案 (Text Readability Solutions)

针对 `FRAGRANT EPIPHANY` 文字在女神像背景上难以辨识的问题，提供以下三种非代码调整方案：

## 方案 A：顶部品牌栏 (Top Brand Header) - **推荐**
*   **做法**：将 `FRAGRANT EPIPHANY` 移至屏幕**最顶部正中**（与语言切换按钮同层级）。
*   **优点**：
    *   彻底避开女神像复杂的线条区域，背景为纯净天空，**绝对清晰**。
    *   符合“杂志/网页”的通用的 Header 布局，增加庄重感。
    *   拉开视觉层次，不与主标题争抢视线。

## 方案 B：磨砂徽章 (Glass Badge)
*   **做法**：为这行文字添加一个**胶囊形的磨砂背景**（半透明白色 + 模糊）。
*   **优点**：
    *   保留在当前位置。
    *   通过背景板强制隔离干扰，保证可读性。
    *   增加精致的 UI 细节。
*   **缺点**：可能会稍微遮挡女神像的面部细节。

## 方案 C：加深与加粗 (Darker & Bolder)
*   **做法**：将文字颜色从“金色”改为“深棕色/黑色”，并增加字重（变粗）。
*   **优点**：
    *   改动最小。
    *   利用色彩对比度解决问题。
*   **缺点**：如果背景线条太密，深色文字可能依然会与线条混淆（“脏”感）。

---
**建议选择方案 A**，既解决了功能问题，又提升了排版的呼吸感。

# UI 视觉与体验优化方案 V3 (UI Optimization Proposal V3)

## 1. 核心反馈与目标
根据您的最新指示，V3 方案将聚焦于以下两点核心调整：
1.  **半球星空 (Hemispherical Star Field)**：在页面底部构建一个完整的、密集的半球形星空背景，而不是散落在全屏。
2.  **按钮布局 (Button Stack)**："Learn More" 文字按钮明确放置在主按钮下方。

---

## 2. 详细设计方案

### A. 布局：天圆地方 (The Starry Dome)
页面将在视觉上被明确分割为“天空”与“星海”两部分。

*   **上部 (Upper Realm - The Goddess)**：
    *   **内容**：女神插画。
    *   **处理**：保持纯净，无任何文字遮挡。女神像居中，作为视觉的主宰。
    *   **背景**：保持暖杏色纯色或极淡的纹理，确保女神脸部和香水瓶清晰可见。

*   **下部 (Lower Realm - The Starry Dome)**：
    *   **形态**：一个巨大的**半球形 (Semi-circle / Dome)** 或 **弧形地平线** 从屏幕底部升起，占据底部约 40-50% 的高度。
    *   **星空填充**：
        *   这个半球体内部是**深邃的星空**（深褐色/深金色渐变）。
        *   **密集星群**：星星只存在于这个半球体内，形成一个“碗状”或“拱门状”的星空容器。
        *   **不散乱**：星星不再是全屏随机分布，而是被约束在这个几何形状内，显得庄重且有秩序。

### B. 内容与交互 (Content & Interaction)
所有文字和交互元素都位于底部的“半球星空”之上或内部。

*   **标题 (Title)**：
    *   悬浮在半球星空的上方边缘，或者位于半球体的上半部分。
    *   颜色：深褐色 `#2B1F16`。

*   **操作区 (Action Stack)** - 位于半球星空的中心位置：
    1.  **主按钮 (Start Explore)**：
        *   位于星空半球的视觉中心。
        *   样式：保留内发光的高级质感。
    2.  **次按钮 (Learn More)**：
        *   **位置**：明确位于主按钮的**正下方**。
        *   **样式**：纯文字链接样式（带下划线或微弱透明度），颜色与星空背景形成高对比（如浅金色 `#F7F2ED` 或深褐色，取决于背景深浅）。

## 4. 实施技术细节
1.  **CSS 形状**：使用 `radial-gradient` 或 `border-radius` 在底部构建一个巨大的半圆容器。
2.  **星空限制**：将 `Sparkles` 组件的 `position` 限制在这个半圆容器内，或者使用 `overflow: hidden` 裁剪。
3.  **Flex 布局**：底部容器采用 `flex-direction: column`，确保按钮垂直排列。

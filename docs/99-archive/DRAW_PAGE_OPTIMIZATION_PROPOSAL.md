# Draw 页面 UI/UX 优化方案 (V6 - 最终定稿)

根据您的最新指示，我们确定了最终的视觉与交互方向：

## 1. 视觉风格 (Visual Style) - **原色调 + 高级素材**
*   **主色调 (Color Tone)**:
    *   **保持原版**: 延续目前的 **深褐色/咖啡色 (Coffee/Dark Brown)** 主色调，不进行大改，保持品牌一致性。
*   **背景升级 (Background)**:
    *   **素材应用**: 使用 `/assets/web` 下的高级纹理图片（如 `07296560-77c6-4ef6-9797-d84d25347096.png` 或其他神秘学风格图片）。
    *   **处理方式**: 对图片进行 **高斯模糊 (Gaussian Blur, 20px+)** 和 **降低透明度 (Opacity 20-30%)** 处理，叠加在深色背景之上。
    *   **效果**: 营造出一种“磨砂玻璃”或“迷雾”后的神秘透视感，比纯色背景更具质感和高级感。

## 2. 交互调整 (Interaction)
*   **取消缩放 (Disable Zoom)**:
    *   移除双指缩放功能，锁定最佳视觉比例。
*   **无限轮播 (Infinite Loop)**:
    *   实现卡牌的无限循环滚动，消除“尽头感”。

## 3. 视觉布局 (Visual & Layout)
*   **卡牌横放 (Horizontal Orientation)**:
    *   恢复卡牌的横向排列（扇面展开）。
*   **C 型布局**:
    *   保持右侧半圆布局，配合横向卡牌，形成优雅的弧线。

## 4. 功能修复 (Fixes)
*   **修复按钮**: 优化中英文切换按钮的位置和显示。

## 5. 总结
**“原汁原味的高级感”**
通过引入项目自带的高质量素材作为环境背景，在不改变原有色调认知的前提下，大幅提升视觉的丰富度和细腻度。

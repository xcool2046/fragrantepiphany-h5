# UI 视觉与体验优化方案 V2 (UI Optimization Proposal V2)

## 1. 问题回顾与核心目标
根据您的反馈，V1 方案（中心玻璃卡片）存在以下严重问题：
1.  **遮挡核心视觉**：女神的脸部和手中的香水瓶被卡片遮挡，破坏了画面的叙事感。
2.  **星空氛围丢失**：柔和的“星尘”过于微弱，导致原本的“星空”主题消失。
3.  **功能入口缺失**："Learn More" 按钮被移除/改为箭头，不符合预期。

**V2 核心目标：** **"让位与重现" (Reveal & Restore)**
把舞台还给女神和星空，文字退守至安全区域，同时强化星空氛围。

---

## 2. 优化方案详情

### A. 布局重构：沉浸式底部重心布局 (Cinematic Bottom Layout)
为了彻底解决“遮挡”问题，我们将放弃中心卡片布局，采用**底部重心**布局。

*   **视觉区域 (Top 60%)**：
    *   **完全留白**：屏幕上部 60% 的区域完全留给女神插画。
    *   **无遮挡**：确保女神的脸庞、手中的香水瓶、上半身轮廓完全清晰，没有任何文字或蒙层覆盖。
    *   **Logo**：仅在顶部保留极小的 "Fragrant Epiphany" 品牌字，或者直接移除，让画面更纯净。

*   **内容区域 (Bottom 40%)**：
    *   **渐变承载**：在屏幕底部添加一个从透明到深色（或深暖色）的柔和渐变蒙层，用于承载文字。
    *   **文字排版**：标题、副标题、按钮全部下沉。
    *   **视觉流**：用户视线先看女神（上），再看文字（下），自然顺畅。

### B. 星空氛围回归 (Starry Sky Restoration)
之前的“星尘”太弱，我们需要找回“星空”的璀璨感，但要避免廉价感。

*   **多层级星空 (Layered Stars)**：
    1.  **背景星尘**：保留 V1 的微弱光点，作为深层背景。
    2.  **前景亮星**：增加 5-8 颗**高亮、带十字星芒**的星星（使用 SVG 或图片），点缀在女神周围（避开脸部）。
    3.  **呼吸动效**：让亮星有明显的“明暗呼吸”效果，增加生命力。
*   **位置策略**：星星主要分布在女神头部周围和画面顶部，形成“众星拱月”的视觉效果。

### C. 功能入口复原 (Learn More Button)
恢复明确的文字按钮，确保入口清晰。

*   **双按钮布局**：
    *   **主按钮 (Start Explore)**：保持 V1 的内发光高级质感，位于底部内容区的核心位置。
    *   **次按钮 (Learn More)**：
        *   恢复为**纯文字按钮**。
        *   放置在主按钮**下方**，或者屏幕最底部的安全区。
        *   样式：使用带下划线或半透明边框的样式，与主按钮区分主次，但保证清晰可见。

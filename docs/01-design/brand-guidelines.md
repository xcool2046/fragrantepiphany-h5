# 品牌与文案基调（来源 Demo_11.18.pptx 摘要版）

## 品牌定位
- 名称：Fragrant Epiphany —— “带着香气的顿悟”。
- 价值主张：塔罗 × 香水，把潜意识状态照亮，并给到“能喷在身上的答案”。
- 体验关键词：温柔、陪伴、个人化；强调香气与心境的联结，而非冷冰冰的测试或清单。

## 语气与风格
- 温柔叙事、意象化：常用月亮/光/庙宇/香气等意象，描述情绪与场景（如“庙堂光柱、尘埃/木料的宁静”）。
- 亲近口吻：用“你”直接对话，提醒“你值得温柔对待自己”。
- 中英文混用可接受，但正式落地建议全英/全中分离，保持一致语气。

## About 核心文案（摘要）
- “一场带着香气的顿悟”，在大改变前点亮温柔转折点。
- 提供：塔罗占卜、香水推荐（100+ 资料库）、独特体验（可喷的答案）。
- 承诺：不只是人格测试/香调清单，而是为“你”挑选的香气，陪伴靠近真实的自己。
- 创办人背景：管理/顾问 + 心理学/音乐/香气爱好，学习塔罗并建立香水库，关注人本与情绪张力。

## 问卷文案（三问四选一）
1) 当前最困扰/思考的人或事：关系与情感｜事业与财富｜自我与成长｜机遇与未知。
2) 最渴望获得：清晰与答案｜勇气与力量｜释放与疗愈｜确认与验证。
3) 目前状态：主动出击｜徘徊观望｜感到耗竭｜冷静分析。

## 抽牌/转场意象
- 抽牌动效：香水滴落/水波纹感的转场（轻量），避免重粒子；半圆轮盘抽牌，第三张后轻量 Loading。

## 推荐文案示例（摘自 PPT，供语气参考）
- 案例：TAM DAO（檀香/乳韵/古老庙宇意象），与“宝剑四/圣杯九/权杖 Ace”对应。
- 语气模版：
  - 气味意象：干燥带绿意的柏树与雪松、禁欲的冷感、千年庙宇的宁静。
  - 塔罗映射：“宝剑四”的宁寂、“圣杯九”的丰盈、“权杖 Ace”的召唤。
  - 标题/标语示例：“于静寂庙堂，寻回内心之光”。
- 适用范围：结果页的推荐/香气描述可沿用此风格，保持简洁、温柔、具象。

## 使用建议
- 前期 UI 文案：可摘取上面 About/语气，用短句呈现；保持中英一致性。
- 推荐/结果文案：控制篇幅（2–3 句），延续“意象 + 指引”风格，避免长段落阻塞 H5。
- 若需更多 PPT 文案，可按需再细化抽取；当前文本足够支撑首版示例与风格对齐。

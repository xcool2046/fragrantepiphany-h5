# UI 优化方案 (最终修正版)

## 核心目标
**回归经典居中构图**，融合现有“高级感”视觉元素，还原原始文案与按钮逻辑。

## 1. 布局重构 (Layout)
*   **回归居中 (Centered Alignment)**：
    *   所有核心内容（标题、副标题、按钮）恢复**水平居中对齐**。
    *   放弃左右分栏，回归更稳重、经典的视觉结构。
*   **女神像处理 (Goddess Image)**：
    *   保持当前的大尺寸和高清晰度。
    *   位置微调至**正中或略微偏上**，作为背景基底。
    *   **关键处理**：在文字显示的区域增加微妙的**光晕遮罩 (Glow Mask)** 或降低局部透明度，确保女神像清晰可见的同时，文字依然易读（避免“文字糊在脸上”）。

## 2. 文案还原 (Content Restoration)
*   **标题 (Title)**：
    *   内容：`A Journey Through Scent, Spirit & Stars`
    *   样式：大号衬线体 (Serif)，居中堆叠排版，保留高级感字间距。
*   **副标题 (Subtitle)**：
    *   内容：`Tarot · Fragrance · You`
    *   样式：位于标题下方，精致衬线体，作为优雅的点缀。

## 3. 按钮设计 (Buttons)
*   **主按钮 (Primary)**：
    *   内容：`Start Explore`
    *   位置：视觉中心下方。
    *   样式：**实心胶囊按钮** (Solid Pill) —— 深色背景 + 金色文字 + 微光泽感。这是最经典且点击率最高的设计。
*   **次按钮 (Secondary)**：
    *   内容：`Learn More`
    *   位置：主按钮正下方 (间距约 24px)。
    *   样式：**纯文字链接** (Text Link) —— 极简风格，带微弱透明度，鼠标悬停加深。

## 4. 氛围营造 (Atmosphere)
*   **星河 (River of Stars)**：
    *   调整为**环绕式**或**放射状**分布，呼应居中构图，不再是对角线切断。
    *   保留忽明忽暗的呼吸效果。
*   **交互 (Interaction)**：
    *   保留鼠标移动时的**视差效果 (Parallax)**，让背景层（女神、星星）与前景文字层产生微妙的位移差，增加深邃感。

---
*此方案旨在平衡“艺术感”与“可用性”，确保在视觉惊艳的同时，信息传达清晰准确。*

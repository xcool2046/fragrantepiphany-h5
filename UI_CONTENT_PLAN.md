# UI 与内容优化方案 (已实施)

## 1. 设计理念
- **核心风格**: "Apple-style" 极简主义与高端感。
- **视觉语言**: 全屏沉浸式体验，无多余滚动，使用高质量背景与磨砂玻璃效果 (Glassmorphism)。
- **交互体验**: 物理动效 (Spring Physics)，手势操作，流畅的页面转场。

## 2. 页面优化详情

### 2.1 首页 (Home)
- **布局**: 全屏固定布局，移除滚动条。
- **视觉**: 使用动态模糊背景 (`bg-home.png`)，营造神秘氛围。
- **交互**: 
  - 标题与副标题采用交错上浮动画 (Staggered Fade-up)。
  - "Start Explore" 按钮增加悬停与点击的弹性缩放效果。
  - 移除 "Highlights" 板块，保持页面纯净。

### 2.2 问卷页 (Quiz)
- **流程**: 采用 "一步一问" (Step-by-step) 的沉浸式体验。
- **进度**: 顶部添加细长的进度条，实时反馈答题进度。
- **交互**: 
  - 选项按钮采用高级金边风格，选中时有光晕效果。
  - "Continue" 按钮仅在选择后浮现，减少视觉干扰。
  - 页面切换使用平滑的淡入淡出与位移动画。

### 2.3 抽牌页 (Draw)
- **核心交互**: 手势驱动的命运之轮 (Gesture-based Wheel)。
  - 用户可拖动轮盘旋转，点击卡牌进行选择。
  - 选中卡牌后飞入左侧卡槽 (Past/Present/Future)。
- **视觉**: 
  - 卡牌背面使用统一的高级纹理 (`pattern.png`)。
  - 选中卡牌翻转动画。
  - 移动端适配为 "水晶球" 风格的悬浮按钮，增加趣味性。

### 2.4 结果页 (Result)
- **布局**: 
  - 桌面端：三栏卡片横向排列。
  - 移动端：纵向堆叠，支持平滑滚动。
- **内容**: 
  - **Past (过去)**: 免费展示，包含卡牌图像与详细解读。
  - **Present/Future (现在/未来)**: 默认锁定，覆盖磨砂玻璃遮罩与锁图标。
- **支付墙 (Paywall)**: 
  - 悬浮式解锁按钮，引导用户支付以查看完整解读。
  - 支付成功后自动解锁内容，并展示 "香水推荐" (Recommendation)。
- **分享**: 顶部导航栏集成分享功能，支持原生分享或复制链接。

### 2.5 其他页面 (Pages)
- **About (关于)**: 品牌故事，全屏背景，优雅排版。
- **How it Works (如何使用)**: 步骤引导，清晰直观。
- **Privacy & Terms (隐私与条款)**: 规范的法律条款页面，保持统一样式。

## 4. 技术栈
- **前端**: React, TypeScript, Tailwind CSS
- **动画**: Framer Motion (用于所有页面转场与微交互)
- **国际化**: i18next (默认强制英文，支持中文扩展)
- **支付**: Stripe (集成 Checkout Session)
